# Methodology

## Objective
Compare host configuration against Microsoft Windows 11 v24H2 Security Baseline.

---

## Step 1: Installed Security Compliance Toolkit
Downloaded and installed the Microsoft Security Compliance Toolkit package.

---

## Step 2: Imported Windows 11 v24H2 Baseline
Imported the baseline templates into Policy Analyzer.

---

## Step 3: Ran Policy Analyzer Comparison
Compared:
- Baseline configuration
- Effective system state

Generated a side-by-side comparison view.

---

## Step 4: Exported Effective State
Exported effective policy configuration for documentation.

---

## Step 5: Identified Deviations
Reviewed highlighted differences between:
- Baseline values
- Current system values

Categorized findings into:
- Critical
- Moderate
- Informational

---

## Step 6: Documented Findings
Recorded deviations, assessed risk, and documented remediation steps.

---

## Conclusion
The process followed a structured configuration assessment methodology 
similar to enterprise baseline validation workflows.
